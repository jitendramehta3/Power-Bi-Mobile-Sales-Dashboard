# Mobile Sales Dashboard | Power BI

## Overview

This project is an interactive Mobile Sales Dashboard built using Power BI.

It provides a clear view of sales performance, customer behavior, and trends

to support business decision-making.


## Key Insights

- Total Sales, Total Quantity, Transactions & Average Price

- MTD vs Same Period Last Year comparison

- Sales trends by Year, Quarter, Month, and Day

- City-wise mobile sales performance

- Brand and Mobile Model-wise analysis

- Payment Method contribution to total sales

- Customer rating distribution


## Tools & Technologies

- Power BI

- Power Query

- DAX (Basic Measures)

- Excel (Data Source)



## Files Included

- Images – Dashboard screenshots


## Key Learnings

- Data cleaning and transformation using Power Query

- Building KPIs and comparison metrics

- Designing business-focused dashboards

- Data storytelling using Power BI
